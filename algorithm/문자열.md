### 1. 문자열 비교

`== 연산자` & `is 연산자`

```python
list1 = [1, 2, 3]
list2 = [1, 2, 3]

list1 == list2
>>> True

list1 is list2
>>> False # 값은 같지만 오브젝트가 별개이기 때문
```

```python
list1 = [1, 2, 3]
list2 = list1 # 같은 오브젝트를 참조함

list1 == list2
>>> True

list1 is list2
>>> True
```
